# big-data-project-ireflix
Big Data analytical methods project using MySQL and Python.

## Project Overview
This repository contains a big data project for the iReflix dataset, including SQL schema/data scripts and analysis support.

## Contents
- `BIG_DATA_PROJECT/DATASET/`: source dataset files and database exports
- `BIG_DATA_PROJECT/SQL/`: SQL initialization scripts, queries, and the new empty SQL file

## Usage
1. Load the SQL scripts into MySQL to create and populate the database.
2. Use Python or SQL tools to run analytics queries on the data.
