## Ireflix Big Data Analytics Project
Big Data analytical methods project using MySQL and Python.

## Project Overview
This repository contains a big data project for the iReflix dataset, including SQL schema/data scripts and analysis support.

## How to Run
1. Execute CA1.sql in MySQL Workbench.
2. Run synthetic_data_generation.ipynb to generate data.
3. Import generated CSV files into MySQL.
4. Run analytical SQL queries.
5. Execute mapreduce_ireflix.ipynb to analyse viewing logs.
